# Sistem-de-Reglare-a-Temperaturii
Termostat digital implementat in Verilog ca proiect de verificare pentru cursul SECI-SI. Modulul citeste temperatura curenta si porneste incalzirea sau AC-ul pana se ajunge la temperatura setata, cu o marja de eroare ca sa nu se tot opreasca si porneasca. Verificarea acopera cazurile de baza, scenarii random si cateva asserturi SVA.
